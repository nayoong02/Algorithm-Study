## 문제 풀이
- game 함수
  - 총 5번의 동작을 하는데, 매 동작 전 백트래킹 가능하도록 map 배열 -> copy 배열에 복사해둠.
  - 동작 후 copy 배열 -> map 배열로 다시로 복구
  - 5번 동작 완료하면 map 배열에서 최대값 찾기 (findMax 함수 호출)
- move 함수
  - dir 값에 따라 다른 동작 진행 
  - `0` : 위, `1` : 아래, `2` : 왼쪽, `3` : 오른쪽
  - `idx`: 넣을 위치, `block`: 최근 블록 수 
- findMax 함수
  - 현재 map 배열에서 최대값 찾기
